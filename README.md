## Description

This is a simple Python project that allows students to register and view grievances. It also provides an admin option to check the total number of grievances.

## Features

* Student Login
* Register Grievance
* View Grievances
* Admin View
* Exit Program

## Technologies Used

* Python
* Command Line Interface (CLI)

## How to Run

1. Install Python
2. Download the project
3. Open terminal in project folder
4. Run the file:

```
python grievance_system.py
```

## Menu Options

1. Login
2. Register Grievance
3. View Grievance
4. Admin View
5. Exit

## Conclusion

This project is a basic example of a grievance system using Python and helps in understanding simple programming concepts.
